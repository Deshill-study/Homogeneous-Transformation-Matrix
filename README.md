# 雅可比矩阵

### 定义与物理意义
雅可比矩阵（Jacobian Matrix）是机器人学中最重要的数学工具之一，它建立了**关节空间**与**操作空间**（笛卡尔空间）之间的微分映射关系。

**核心思想**：描述关节的微小变化如何引起末端执行器位姿的变化。

对于n自由度机械臂，设：

- **q** = [q₁, q₂, ..., qₙ]ᵀ：关节变量（旋转关节为角度，移动关节为位移）
- **x** = [x, y, z, α, β, γ]ᵀ：末端执行器位姿（位置和姿态）

则位姿与关节变量的关系为：
```
x = f(q)
```

这里的变化就是将关节数量维度通过f映射为末端执行器位姿维度,注意f是一个向量函数
```
R^n -> R^6
```
雅可比矩阵定义为 **f** 对 **q** 的偏导数：
```
J(q) = ∂f/∂q = [∂x/∂q₁  ∂x/∂q₂  ...  ∂x/∂qₙ]
               [∂y/∂q₁  ∂y/∂q₂  ...  ∂y/∂qₙ]
               [∂z/∂q₁  ∂z/∂q₂  ...  ∂z/∂qₙ]
               [   ...      ...  ...      ...]
               [∂γ/∂q₁  ∂γ/∂q₂  ...  ∂γ/∂qₙ]
```
#### 速度映射关系

雅可比矩阵的核心作用是建立**速度映射**：

```
ẋ = J(q) · q̇
```

其中：
- **ẋ** = [v_x, v_y, v_z, ω_x, ω_y, ω_z]ᵀ：末端执行器速度（6×1）
  - [v_x, v_y, v_z]：线速度（m/s）
  - [ω_x, ω_y, ω_z]：角速度（rad/s）
- **q̇** = [q̇₁, q̇₂, ..., q̇ₙ]ᵀ：关节速度（n×1）
- **J(q)**：6×n 雅可比矩阵

**直观理解**：J 的每一列表示"当第i个关节以单位速度运动时，末端执行器产生的速度"。

#### 雅可比矩阵结构

```
        关节1    关节2    ...    关节n
     ┌─────────────────────────────────┐
v_x  │ ∂x/∂q₁  ∂x/∂q₂  ...  ∂x/∂qₙ │
v_y  │ ∂y/∂q₁  ∂y/∂q₂  ...  ∂y/∂qₙ │  ← 线速度部分 J_v (3×n)
v_z  │ ∂z/∂q₁  ∂z/∂q₂  ...  ∂z/∂qₙ │
     ├─────────────────────────────────┤
ω_x  │   ...      ...  ...      ...  │
ω_y  │   ...      ...  ...      ...  │  ← 角速度部分 J_ω (3×n)
ω_z  │   ...      ...  ...      ...  │
     └─────────────────────────────────┘
```

### DH四个参数的全称
DH 一共 4 个参数，用来描述两个相邻连杆坐标系的关系：
1. θᵢ (theta)英文：Joint angle中文：关节角含义：绕前一个坐标系 Z 轴的旋转角
2. dᵢ英文：Link offset中文：连杆偏距含义：沿前一个坐标系 Z 轴的平移
3. aᵢ英文：Link length中文：连杆长度含义：沿当前坐标系 X 轴的平移
4. αᵢ (alpha)英文：Link twist angle中文：连杆扭转角含义：绕当前坐标系 X 轴的旋转角

## 雅可比矩阵的常用求解方法
机器人学中求解雅可比矩阵主要有三种实用方法，分别适用于不同场景，其中向量叉乘法是工程代码（如你提供的 Python 程序）最常用的实现方式：
#### 方法一：解析求导法（直接微分法）
适用场景：平面机械臂、2/3 自由度简单机械臂，推导直观、公式清晰核心步骤：
先建立机械臂正运动学方程，写出末端位姿 x=f(q), y=f(q), z=f(q) 关于关节变量的显式表达式
对每个关节变量 qᵢ 分别求偏导数
将所有偏导数按列拼接，形成雅可比矩阵
以 2 关节平面机械臂为例：正运动学：x = L₁cosθ₁ + L₂cos(θ₁+θ₂)y = L₁sinθ₁ + L₂sin(θ₁+θ₂)分别对 θ₁、θ₂ 求偏导，直接得到 2×2 雅可比矩阵。
#### 方法二：向量叉乘法（几何矢量法，代码标准实现）
适用场景：3D 空间、6 自由度多关节机械臂，适配 DH 参数，是你提供代码的核心逻辑  
核心原理：旋转关节的运动通过叉乘映射为末端线速度，角速度直接取关节旋转轴  
完整求解步骤：  
1. 通过 DH 变换矩阵，从基座到末端逐关节递推，计算每个关节坐标系的：
    1. 原点位置向量 pᵢ（基坐标系下）
    2. 旋转轴单位向量 zᵢ（默认绕 Z 轴转动，取坐标系 Z 轴）
2. 确定末端执行器在基坐标系下的位置 pₑₙd
3. 逐列构建雅可比矩阵：
    1. 线速度列（前 3 行）：J_v[:,i] = zᵢ × (pₑₙd - pᵢ)
    2. 角速度列（后 3 行）：J_ω[:,i] = zᵢ
4. 合并线速度与角速度部分，得到最终 6×n 雅可比矩阵
特点：无需显式求导，直接通过坐标变换计算，适配任意多自由度机械臂，工程落地性最强。
